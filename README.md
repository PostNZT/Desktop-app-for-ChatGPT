## 🏁 Getting Started <a name = "getting_started"></a>

### Prerequisites

- Node.js
- npm

### Installing

1. Clone the repo

```bash
git clone
```

2. Install dependencies

```bash

npm install
```

3. Run the app

```bash
npm start
```
